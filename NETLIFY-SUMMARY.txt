🚀 DEPLOYMENT METHODS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

METHOD 1: DRAG & DROP (Fastest - 30 seconds)
────────────────────────────────────────────
1. Visit: https://app.netlify.com/drop
2. Drag /docs folder into browser
3. Wait for deployment
4. Change site name to "brennan-evolveu"
5. Done!

METHOD 2: GITHUB INTEGRATION (Recommended - Auto-deploy)
─────────────────────────────────────────────────────────
1. Push to GitHub:
   $ git add .
   $ git commit -m "Add Netlify config"
   $ git push origin main

2. Connect on Netlify:
   - Go to app.netlify.com
   - "Add new site" → "Import project"
   - Choose GitHub → "brennanbrown/evolveu"
   - Settings auto-detected from netlify.toml
   - Click "Deploy"

3. Set site name to "brennan-evolveu"

4. Every future git push = automatic deployment! 🎉
